Version history and Road map

Version history

V. 0.2 (Dec 6, 2015)

  • Release status change for StyleClassChangeRenderKit, <cf:formLayout> and <cf:readOnly> from „Proof of concept“ to Work in progress.
  • Support for all basic PrimeFaces input components: <p:inputText>, <p:inputTextarea>, <p:selectBooleanCheckbox>, <p:selectOneMenu>, <p:calendar> for StyleClassChangeRenderKit, <cf:formLayout> and <cf:readOnly>.
  • <cf:formLayout>: Support for a:colspan attribute; small enhancements.
  • New component: @MaxSize validator.
  • All “major” bugs fixed.
  • Minor bugfixes.
See the detailed issue tracker here.

V. 0.1 (Nov 15, 2015)

  • First stable release, including "proof of concepts" for core functionality.

Road map

V. 0.3

Future releases

There will probably be a few more 0.x releases until the version jump to 1.0 once a certain degree of “feature completeness” is reached.
